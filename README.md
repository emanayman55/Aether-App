# Aether-App
🌦️ Aether - Real-time Weather Dashboard
Aether is a modern, responsive, and minimalist web application designed to provide real-time weather updates with a focus on clean user experience (UX) and visual clarity.
✨ Key Features
• Minimalist UI: A sleek dark-themed interface with smooth gradients and rounded-corner cards for a premium feel.
• Real-time Data: Instant access to current temperature, humidity, wind speed, and visibility.
• Global Search: Ability to fetch weather data for any city worldwide with a single click.
• Smart Error Handling: Implemented robust try...catch blocks and JSON validation to ensure the app stays alive even if the API data is malformed.
🛠️ Technical Stack
• Frontend: HTML5 & CSS3 (using Modern Flexbox and CSS Variables).
• Scripting: JavaScript (ES6+).
• Data Format: JSON (JavaScript Object Notation) for API communication.
• Error Management: Global error handling via window.onerror and manual throw operators for incomplete data.
